  ## Layout Structure Summary

  **Fixed Elements:**
  - **Sidebar Width:** 250px fixed
  - **Header:** "CM [X]" with app title and close button
  - **Navigation:** Accessible via CM button (opens/closes, home navigation)

  **Content Area:**
  - **Scrollable:** Vertical scroll when content exceeds viewport
  - **Responsive:** Elements scale down to fit 250px width
  - **Theme Support:** Dark/light mode theming throughout

  **Interaction Patterns:**
  - **CM Button:** Toggle open/close, navigate home when open
  - **Navigation:** Route-based navigation between screens
  - **Form Submission:** Direct integration with LeetCode problem data
  - **Session Management:** Timer and problem tracking workflows

  This markdown layout shows the visual structure and content organization of
  each screen in your Chrome extension's content script overlay that appears on       
  LeetCode pages.
